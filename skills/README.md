# Skills Overview

本仓库的 skills 主要集中在 `skills/` 目录下（按类别分文件夹）。

## How to use

- 把你的需要的某个`Github Skill链接` 发给Cursor、Openclaw 或 Hermes，然后让他自己安装。举例：
- `安装这个skill https://github.com/chenyueli829/skillhub-public/blob/main/skills/%E7%A0%94%E7%A9%B6%E7%B1%BB/ecc-deep-research/SKILL.md`
