# iWiki 知识库整理

从 iwiki 拉取文档到本地：自动下载正文、图片（带序号）、附件，并记录文档里的腾讯文档链接。

## 怎么用

对 AI 说「拉取」并给出 iwiki 链接即可。

- **空间链接**：`https://iwiki.woa.com/space/空间key`（会拉取整个空间）
- **文档链接**：`https://iwiki.woa.com/p/文档id`（单篇或整棵子树）

示例：`拉取 https://iwiki.woa.com/p/4015651258`

保存位置可沿用默认（见下），也可指定目录。

## 默认保存位置

| 类型 | 路径 |
|------|------|
| 文档 | `我的知识库/iwikis/`（按「空间 / 文件夹 / 文档.md」层级） |
| 图片 | **同一级**目录下共用的 `_images` 文件夹，各文档在 `_images/{文档名}/` 下（如 `_images/API设计/1-图.png`） |
| 附件、腾讯文档引用 | **同一级**目录下共用的 `_files` 文件夹，各文档在 `_files/{文档名}/` 下 |

目录会按「空间 / 文件夹 / 文档」层级建好；**路径里的名称会去掉空格**（如 `0 整体视图` → `0整体视图`）。同一级多个文档共用 `_images`、`_files` 两个固定名称的文件夹，按文档名再分子目录，避免重名。

## 几条规则（摘要）

- 图片文件名：`序号-原名.扩展名`（如 `1-架构图.png`）。
- 文档里引用的腾讯文档会生成小文件：`编号-腾讯文档-文档名.md`，内容只有「来源文档」和「链接」。
- 详细规则都在 **SKILL.md** 里，AI 会按那里执行。

## 常见问题

- **要登录吗？** 需要能访问 iwiki（如已登录 OA/企业微信），MCP 会处理鉴权。
- **支持哪些链接？** 空间 URL 和文档 URL（`/p/数字`）。其它格式会提示错误。
- **更多示例 / 历史文档？** 见根目录 **docs_archive/**。
