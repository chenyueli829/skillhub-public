# 腾讯文档引用文件 - 格式说明

## 📋 文件命名

**命名规则**：`{编号}-{文档来源}-{文档名称}.md`

- 例如：`1-腾讯文档-资金账务业务发展规划.md`
- 多个链接时：`1-腾讯文档-名称1.md`、`2-腾讯文档-名称2.md`（每个链接一个文件），或同一文件内列出多条链接、文件名用第一个文档名称。
- **不再使用**固定文件名 `关联的腾讯文档.md`。

---

## 📂 文件位置

路径中的**文件夹名、文件名均去除空格**（不添加其他符号）。

```
我的知识库/
└── files/
    └── {空间名}/           ← 名称去空格
        └── {文件夹路径}/   ← 各段去空格
            └── {文档名}/   ← 去空格
                ├── 1-腾讯文档-文档名称A.md
                └── 2-腾讯文档-文档名称B.md
```

---

## 📝 文件内容（简化）

内容只保留两要素：**来源文档**、**链接**。不写长说明、注意事项、类型枚举等。

### 简化模板

```markdown
# {编号}-腾讯文档-{文档名称}

来源文档：[{来源 iwiki 文档标题}]({相对路径到该 .md})

**链接**：{腾讯文档完整 URL}
```

### 示例

```markdown
# 1-腾讯文档-资金账务业务发展规划

来源文档：[资金账务领域架构](../../../iwikis/资金账务/0整体视图/资金账务领域架构.md)

**链接**：https://doc.weixin.qq.com/slide/p3_AMAAxgbdAFwzr5MI0i6TXKZfSL9sr?scode=...
```

若同一文件内记录多个链接，可依次增加「**链接**」行；或为每个链接单独一个文件（文件名用对应文档名称）。

---

## 📈 在 iwiki 文档中的引用

### Frontmatter

```yaml
external_dependencies: ../../../files/{space}/{folder}/{doc}/1-腾讯文档-{文档名称}.md
```

路径中的 space、folder、doc 均为去空格后的名称。
